## 🚀 كيفية التشغيل

### الخيار الأول: بدون تثبيت أي شيء (الأسهل) ⭐

1. افتح `RUN.bat` مرتين متتاليتين
2. أو افتح المجلد وانقر على `public/index.html`

### الخيار الثاني: مع Python (موصى به)

**تأكد من تثبيت Python أولاً:**
https://www.python.org/downloads/

ثم قم بـ:
```bash
# في مجلد المشروع
python -m http.server 8000
```

ثم افتح في المتصفح:
```
http://localhost:8000
```

### الخيار الثالث: مع Node.js (الكامل مع قاعدة بيانات)

**تثبيت Node.js أولاً:**
https://nodejs.org/

ثم:
```bash
# تثبيت المتطلبات
npm install

# تشغيل الخادم
npm start

# افتح المتصفح على:
http://localhost:3000
```

## 📋 خطوات الاستخدام

### للمستخدم العادي:

1. **التسجيل:**
   - اختر "إنشاء حساب"
   - أدخل اسم مستخدم وكلمة مرور
   - اضغط "إنشاء حساب"

2. **تسجيل الدخول:**
   - اختر "تسجيل دخول"
   - أدخل بيانات حسابك
   - اضغط "دخول"

3. **إنشاء عملية وساطة:**
   - اضغط زر "➕ إنشاء عملية"
   - اختر نوع السلعة
   - أدخل السعر
   - اختر دورك (بائع/مشتري)
   - اضغط "✓ موافق"

4. **الشات:**
   - اضغط "💬 فتح الشات"
   - أرسل رسالتك
   - ستجد رقم التذكرة فوق الشات

### للمسؤول:

1. **تسجيل الدخول:**
   - Username: `m5tl`
   - Password: `m5tl123`

2. **إدارة التذاكر:**
   - اذهب إلى "⚙️ لوحة التحكم"
   - شاهد جميع التذاكر
   - افتح الشات مع أي تذكرة
   - أغلق التذاكر المكتملة

## 🆘 استكشاف الأخطاء

### المشكلة: الصفحة بيضاء
**الحل:** 
- تأكد من أن جميع الملفات في مجلد `public`
- جرب `Ctrl + Shift + R` لتحديث الصفحة بالكامل

### المشكلة: الشات لا يعمل
**الحل:**
- تأكد من أن localStorage مفعل في المتصفح
- جرب متصفح آخر

### المشكلة: Discord webhook لا يعمل
**الحل:**
- هذا اختياري، الموقع يعمل بدونه
- تأكد من الاتصال بالإنترنت
- جرب خادم Node.js للعمل الكامل
